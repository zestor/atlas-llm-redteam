# Lab Solutions

Reference implementations for all 12 labs. These are intentionally kept in a separate directory so practitioners can complete the labs before reviewing solutions.

## Usage Policy

- Complete the lab yourself first — the learning is in the struggle
- Reference solutions only after attempting the `TODO:` markers in each lab's skeleton code
- Each solution includes commentary explaining the design decisions
