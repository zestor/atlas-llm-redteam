## Expertise Levels

This wiki is structured for a spectrum from practitioner to researcher.

**Practitioner** — Understand attack categories, run the scanner, complete Labs 01–03.

**Intermediate** — Implement PAIR, TAP, and system prompt extraction. Complete Labs 04–06.

**Expert** — MCP attack chains, multi-agent compromise, agent trust analysis. Complete Labs 07–09.

**Researcher** — Mechanistic interpretability attacks, sleeper agent detection, novel dataset creation. Complete Labs 10–12.
